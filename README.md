# ES6 Pathfinder

Using the A* Algorithm with Manhattan Heuristics.

## Prerequisites

Ensure you have `node` installed.

## Running Example

`npm run index`

## Running Tests

Ensure you have Jasmine installed globally `npm i -g jasmine`

`npm run test`
